# 📈 Profit Analysis Using Multiple Linear Regression (Orange)

## 📖 Overview

This project uses **Orange Data Mining** to build a Multiple Linear Regression model for predicting company profit based on business expenditures such as **R&D Spend**, **Administration**, and **Marketing Spend**.

The project covers the complete regression workflow, including data preprocessing, model building, performance evaluation, and visualization. It demonstrates how regression analysis can be used to understand the relationship between company investments and profit while generating actionable business insights.

---

## 🎯 Objectives

- Analyze the factors affecting company profit.
- Build a Multiple Linear Regression model using Orange Data Mining.
- Evaluate model performance using regression metrics.
- Visualize relationships between variables.
- Provide business recommendations based on the results.

---

## 🛠️ Tools Used

- Orange Data Mining
- CSV Dataset
- Multiple Linear Regression
